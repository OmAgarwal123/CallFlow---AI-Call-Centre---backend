# CallFlow Backend

Multi-tenant AI voice call center backend built with Node.js, Twilio, OpenAI, Redis, and Docker.

## Features
- AI voice agent
- Multi-tenant SaaS architecture
- Redis-backed memory
- Call analytics
- Rate limiting & abuse protection
- Dockerized deployment

## Setup
1. Clone repo
2. Create `.env` from `.env.example`
3. `docker-compose up --build`
